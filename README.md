# Backend Node

Este documento te ayudara a iniciar el proyecto de manera correcta, sigues los pasos descritos abajo.

## Configuración del proyecto

```sh
pnpm install
```

### Compilar para desarrollo

```sh
pnpm run dev
```

### Compilar para producción

```sh
pnpm run start
```
